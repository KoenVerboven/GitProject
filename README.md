Git commandline :
Visual studio, choose View, choose Terminal.

Get version:

git --version

Get all branches:

git --branch

Create new branch:

git branch feature_medicalworker

shorter :

git checkout -b feature_medicalworker

Switch between branches:

git checkout feature_medicalworker

Show all commits:

git log
